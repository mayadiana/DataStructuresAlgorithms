# Data Structures and Algorithms - C++ Implementations

**A collection of Abstract Data Types (ADTs) implemented from scratch in C++.**

This repository contains various data structures developed during my Computer Science studies. Every implementation focuses on memory efficiency, algorithmic complexity, and low-level pointer management.

## 🛠️ Key Technical Highlights
* **Manual Memory Management:** All structures use dynamic allocation (`new`/`delete`) without relying on STL containers like `std::vector` or `std::list`.
* **Custom Iterators:** Every ADT includes a custom iterator implementation for efficient traversal.
* **Algorithm Efficiency:** Focused on achieving optimal time complexity for basic operations (insertion, deletion, search).
